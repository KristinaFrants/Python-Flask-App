About This App

Thats a basic social network Flask app. User is abble to Register, Login, Logout, Post, Edit and Delete articles. MYSQL Dolphin is used to communicate with a database.

Bugs that have been faced while buildinf app:


  • Because no SQLAlchemy is used, I had manually to implement all the requirements for MYSQl and match it with .env file
  • While installing pip, requirements.txt were conflicting with pipfile, so I had to remove pipenv complitelly
  • When deployed to Heroku had to import os and change MYSQL config to match .env and Heroku JawDB for MYSQL
  • Most of the small bugs were misspeling or missed comma in functions or routes
  • For each route an HTML template is created, most components are returned with render_template and helpers
